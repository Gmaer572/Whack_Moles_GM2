# Whack a Mole

A fully functional remake of Whack A Mole in GameMaker Studio 2. 

## Controls/Rules:
Control using the mouse, and click on mole to whack them. You have thirty seconds to hit as many moles as you can within the time limit. **High scores are not saved.**

Also, please note there is NO RESOLUTION SCALING, among other things. This was made **purely as a practice** and may be subject to a lack of polish in certain areas, especially in the graphics department.

## Build/run Instructions:
1. Click the "code" button on this page, then "Download as ZIP". (You can also clone it with GitHub Desktop if you know how to use it.)

2. Once downloaded, go into the folder labelled "whack a mole" and open the "whack a mole.yyp" file with GameMaker Studio 2. (You may need to update your IDE).

3. Run the game by pressing F5, or by hitting the triangle button labelled "Run" near the top of the screen.
